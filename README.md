# 文章社群文案與圖像自動生成技能 (Article Social Media and Image Generation Skill)

本技能旨在自動化處理「大眾文章需求表」中的內容，為指定的文章產生社-群媒體文案和配圖，並將結果回填至指定的 Google Sheets 中，以簡化內容營運流程。

## 核心功能

這個技能會執行以下一系列自動化步驟：

1.  **讀取歷史紀錄**：首先，技能會檢查「n8n產圖記錄表」，根據已交稿的文章連結和主題，確保不會重複處理已經完成的內容。
2.  **判斷當前任務**：
    *   技能會根據今天的日期，自動定位到「2026大眾文章需求表」中對應的月份頁籤（例如，4月2日會尋找 `2026.04` 頁籤）。
    *   接著，它會篩選出 B 欄中標示使用產品為「籌碼K線」或「起漲K線+籌碼K線」的文章作為待處理任務。
3.  **複製需求主題**：將符合條件的文章需求主題（C 欄）複製到「籌K自動化-2.0 的副本」工作表中最新的空白列。
4.  **讀取文章內容**：點擊「2026大眾文章需求表」L 欄的交稿網址，進入 Google Doc 並複製全文。
5.  **生成社群文案**：
    *   登入指定的 Google 帳號。
    *   使用專屬的 Gemini Pro 模型（文案 Gem），貼上文章內容以生成三組社群文案。
    *   將這三組文案貼回到「籌K自動化-2.0 的副本」對應的 F 欄和 S 欄（共三列）。
6.  **生成繪圖指令**：
    *   使用專屬的 Gemini Pro 模型（繪圖指令 Gem），貼上相同的文章內容以生成三組繪圖提示詞（Prompt）。
7.  **生成與處理圖片**：
    *   使用 Gemini 圖像生成模型，根據上述三組提示詞分別生成三張圖片。
    *   自動判斷每張圖片的背景色（深色或淺色）。
    *   根據背景色，為圖片疊加對應的 Logo（深色背景疊加白色 Logo，淺色背景疊加深色 Logo）。
8.  **上傳圖片並取得連結**：
    *   將處理好的三張圖片上傳至內部圖床。
    *   獲取每張圖片的公開存取連結。
9.  **回填圖片連結**：將三張圖片的連結分別貼回到「籌K自動化-2.0 的副本」中對應的 G 欄和 I 欄（同一張圖的連結會填入同一列的 G 和 I 欄）。

透過這個自動化流程，可以大幅減少手動複製貼上、生成內容和管理檔案所需的時間，確保內容產出的一致性與效率。
